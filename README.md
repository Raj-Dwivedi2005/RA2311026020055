# Vehicle Maintenance Scheduler

## 📌 Description
Backend system to optimize vehicle maintenance scheduling using dynamic programming (Knapsack Algorithm) under limited mechanic hours.

---

## 🚀 Features
- Logging Middleware using external API
- Vehicle scheduling using Knapsack optimization
- REST API integration (Depots & Vehicles)
- Notification system design (6 stages)

---

## 🛠 Tech Stack
- Node.js
- Axios

---

## 📁 Project Structure
- logging_middleware/ → reusable logging function
- vehicle_maintenance_scheduler/ → scheduling logic
- notification_system_design.md → system design (Stage 1–6)

---

## ⚙️ How to Run
```bash
npm install
node vehicle_maintenance_scheduler/index.js

---

Output
Calculates maximum impact for each depot based on available mechanic hours
Logs all operations using custom logging middleware
